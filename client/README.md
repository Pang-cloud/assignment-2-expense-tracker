# Expense Tracker — Frontend

React-based single-page frontend for the Expense Tracker application.

## Requirements

- Node.js 16+
- Backend server running on `http://localhost:5001` (see `../server/`)

## Setup

```bash
npm install
npm start
```

Opens automatically at `http://localhost:3000`.

> Make sure the backend server is running first, otherwise data will fail to load.

## Available Scripts

| Script | Description |
|--------|-------------|
| `npm start` | Start the development server |
| `npm run build` | Build for production |
| `npm test` | Run tests |

## Key Dependencies

| Package | Purpose |
|---------|---------|
| React 19 | UI framework |
| Ant Design 6 | Component library |
| Axios | HTTP client for API calls |
| moment.js | Date handling for DatePicker |
